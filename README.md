# Linux for DevOps 🚀

This repository contains Linux concepts, commands, and troubleshooting topics required for DevOps Engineers.

It covers real-world scenarios used in production environments and DevOps interviews.

- Linux Basics
- Linux Commands
- File System
- User & Group Management
- Permissions
- Process Management
- Disk Management
- Networking
- Package Management
- Systemd Services
- Shell Scripting
- Linux Troubleshooting

## Goal

This repo helps DevOps engineers:

- Prepare for DevOps interviews
- Troubleshoot production servers
