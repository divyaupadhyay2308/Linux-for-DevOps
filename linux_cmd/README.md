## 💻 Basic Linux Commands

### System Information

```bash
uname -a
```

### Check hostname

```bash
hostname
```

### Current logged-in user

```bash
whoami
```

### Current date and time

```bash
date
```

### System uptime

```bash
uptime
```

---

## 📁 Basic File System Navigation

### List files and directories

```bash
ls
```

### Show current directory

```bash
pwd
```

### Change directory

```bash
cd /path
```

### Create directory

```bash
mkdir folder_name
```

### Remove directory

```bash
rmdir folder_name
```

---

## 📄 File Operations

### Create file

```bash
touch file.txt
```

### View file content

```bash
cat file.txt
```

### Copy file

```bash
cp file.txt backup.txt
```

### Move or rename file

```bash
mv file.txt newfile.txt
```

### Delete file

```bash
rm file.txt
```

---

## ❓ Getting Help in Linux

Linux provides built-in help for commands.

### Manual page

```bash
man command_name
```

Example:

```bash
man ls
```

### Command help

```bash
command --help
```
